# Prompts para Gemini (Logo y Portada)

Uso sugerido: pegar el prompt completo en Gemini, generar varias variantes y elegir la mejor.

## Prompt 1 - Logo oficial de SENA / ALERTA-ED

```text
Disena un logo profesional y memorable para un proyecto edtech llamado "ALERTA-ED", cuya mascota IA se llama "SENA".

Contexto:
- Sistema de alerta temprana para prevenir desercion escolar.
- Imagen de marca: seria, humana, tecnologica, confiable.
- El logo debe funcionar en GitHub, presentaciones, favicon y portada de proyecto.

Direccion visual:
- Estilo: minimalista, geometrico, premium, limpio.
- Concepto: combinar un simbolo de proteccion/alerta + educacion + inteligencia artificial.
- Evitar logos genericos de "cerebro con circuitos" o iconos saturados.
- Incluir una forma principal fuerte (ej: hexagono, escudo o nodo) con un detalle distintivo.

Paleta:
- Fondo oscuro opcional: #04080F
- Azul principal: #3B82F6
- Indigo de apoyo: #6366F1
- Acento alerta: #F59E0B
- Blanco azulado para contraste: #EFF6FF

Tipografia:
- Sans moderna y legible.
- Version con texto: "SENA" y subtitulo pequeno "ALERTA-ED" (opcional).
- Entregar tambien una version solo isotipo (sin texto).

Entregables:
1) Logo horizontal (isotipo + wordmark)
2) Logo compacto (solo isotipo)
3) Version positiva y negativa (fondo claro y oscuro)
4) PNG con fondo transparente y alta resolucion
5) Version centrada para avatar

Calidad:
- Bordes nitidos, composicion equilibrada, alto contraste, look startup profesional.
- No usar mockups 3D, no sombras exageradas, no texturas de papel.
```

## Prompt 2 - Portada de GitHub (Social Preview / Cover)

```text
Crea una portada horizontal premium para GitHub del proyecto "SENA - ALERTA-ED".

Formato:
- Aspect ratio: 16:9 (1280x640 recomendado)
- Exportar PNG de alta calidad
- Composicion limpia para verse bien en miniatura

Objetivo:
- Mostrar que es una plataforma de alerta temprana educativa con IA.
- Debe sentirse tecnologica, institucional y moderna.

Direccion visual:
- Fondo oscuro con gradientes sutiles y geometria abstracta (sin ruido excesivo).
- Elementos visuales sugeridos: nodos conectados, paneles de datos estilizados, trazas de flujo.
- Incluir un icono/simbolo central coherente con el logo de SENA.

Texto dentro de la portada:
- Titulo principal: "SENA - ALERTA-ED"
- Subtitulo: "Sistema de alerta temprana para prevenir la desercion escolar"
- Tagline pequeno: "IA + intervencion preventiva + impacto educativo"

Paleta:
- Base: #04080F y #080E1A
- Primario: #3B82F6
- Secundario: #6366F1
- Acento: #F59E0B
- Texto: #EFF6FF y #7CB3E0

Restricciones:
- No estilo caricatura.
- No fotos de stock.
- No iconografia infantil.
- No saturar con demasiado texto.

Resultado esperado:
- Portada elegante, legible, de alto impacto visual, ideal para perfil publico de GitHub y presentaciones.
```

## Prompt 3 - Variante de portada sin texto (solo visual)

```text
Genera una portada abstracta y elegante para un proyecto edtech de IA, sin texto, usando una paleta azul oscuro con acentos ambar.
Debe sugerir deteccion temprana, analitica educativa y proteccion del estudiante.
Composicion minimalista, profesional, limpia, apta para GitHub social preview en 16:9.
```
